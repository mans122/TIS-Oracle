=============문자열==============
char : 고정길이 문자열
varchar2 : 가변길이 문자열

char(5) : 부족한자리는 space로 채움
=> 'abc' >> 'abc  '
=> 주민등록번호,우편번호 등 길이가 일정한 데이터에 사용하면 속도가 빠름

varchar2(5) : 부족한 자리를 space로 채우지 않음
=> 'abc' >> 'abc'
=> 길이가 일정하지 않은 주소,이름 등 에 사용

==============숫자==============
NUMBER(5,3) : 전체5자리, 소수점 3자리 표시
NUMBER(5) : 전체5자리
FLOAT : 실수

==============날짜==============
DATE : 날짜 표시
TIMESTAMP : 밀리초까지 표현가능 / 정확도가 DATE보다 더 높다
